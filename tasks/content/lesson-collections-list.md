## Описание

Создать урок о List - основной immutable коллекции в Scala. Студенты изучат:

- Базовые методы (head, tail, isEmpty, length)
- Практическую работу со списками

Это **второй урок из MVP** - необходим для понимания map/filter в следующем уроке.

## Заметки

### Критические аспекты
- **Визуализация cons-структуры** - ключ к пониманию
- **Связь с immutability** - почему это важно
- **Постепенность** - не перегружать деталями реализации

### Важно показать
```scala
// Cons структура
val list = 1 :: 2 :: 3 :: Nil
// эквивалентно
// 1 :: (2 :: (3 :: Nil))
// визуально: [1] -> [2] -> [3] -> Nil
```

### Избежать
- Углубления в производительность операций (это урок 6)
- Сложных рекурсивных примеров (оставить для ФП раздела)
- Упоминания fold/reduce (это урок 4)
