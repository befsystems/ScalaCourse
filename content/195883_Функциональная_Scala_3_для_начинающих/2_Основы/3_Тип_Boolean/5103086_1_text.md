# Тип данных Boolean

Тип **Boolean **представляет логический тип данных, который может принимать два значения: `true` (истина) или `false` (ложь). Логические значения используются для выполнения условных операций и принятия решений в программе.

![George Boole.jpg](https://upload.wikimedia.org/wikipedia/commons/thumb/6/6c/George_Boole.jpg/274px-George_Boole.jpg)

Название типа `Boolean` получило в честь английского математика и логика [Джорджа Буля](https://ru.wikipedia.org/wiki/%D0%91%D1%83%D0%BB%D1%8C,_%D0%94%D0%B6%D0%BE%D1%80%D0%B4%D0%B6), среди прочего занимавшегося вопросами математической логики в середине XIX века.

Тип `Boolean` обеспечивает булеву алгебру и работу с логическими выражениями.

[Булева алгебра](https://ru.wikipedia.org/wiki/%D0%91%D1%83%D0%BB%D0%B5%D0%B2%D0%B0_%D0%B0%D0%BB%D0%B3%D0%B5%D0%B1%D1%80%D0%B0) (или алгебра логики) — это математическая дисциплина, основанная на работах Джорджа Буля. Она изучает булевы значения и операции над ними. Булева алгебра нашла широкое применение в электронике, компьютерных науках, логике, и других областях.

Тип `Boolean` имеет два литерала: `true` и `false`. Литералы - это явные значения типа, которые можно использовать в коде программы.

При наименовании значений типа `Boolean` принято использовать приставку **is** или **has**, чтобы значение как бы отвечало на вопрос.

Пример:

val isCorrect = true // это правильно?
val hasSense = false // имеет смысл?
