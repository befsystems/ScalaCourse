# Примитивные типы данных

**Почему эти типы данных называются "примитивные"?**

Это **базовые** типы данных представляют множества простейших **значений** в языке, такие, например, как числа, символы или логические значения. Соединяя их разными способами в структуры, можно создавать **сложные типы** данных. Например, соединяя символы, получаем сложный тип - строку символов.

Считается, что примитивные типы уже невозможно разделить на составные части с помощью конструкций языка, это неделимые сущности.

## Свойства примитивных типов

- **Неизменяемость**: эти типы неизменяемы, то есть их значение не может быть изменено после создания.

- **Простота**: примитивные типы имеют **фиксированную область значений, поведение и набор операций**, что делает их простыми для использования и эффективными в вычислениях.

**Перечень примитивных типов данных:**

- **Int** - целые числа (42, `-7`)

- **Long** - длинные целые числа (`1234567890123L`, `-9876543210L`)

- **Short** - короткие целые числа (`32000.toShort`, `-10000.toShort)`

- **Byte** - очень короткие целые числа (`100.toByte`, `-128.toByte)`

- **Float** - числа с дробной частью (`3.14f`, `-0.001f)`

- **Double** - точные числа с дробной частью (`3.141592653589793`, `-2.718281828459045)`

- **Char** - текстовый символ (`'A'`, `'z'`, `'\n')`

- **Boolean** - логическое значение (`true`, `false)`

- **Unit** - пустое множество ().

Все примитивные типы данных - потомки родительского типа **AnyVal** (тип-значение).

Хотя примитивных типов немного, их **комбинации **позволяют создавать множество сложных типов данных. Кроме того, помимо уже существующих стандартных примитивных типов, вы можете создавать **свои** примитивные типы.

Особенность примитивных типов состоит в том, что они часто используются, поэтому, работа с ними хорошо **оптимизирована** на уровне компилятора и виртуальной машины.

## Названия типов

Все типы данных имеют свое **уникальное **название, и оно должно начинаться с **большой** буквы. Например: **Int**

## **Операции типов данных**

Для всех типов данных, включая примитивные, **определены допустимые операции. **Хотя названия операций могут быть одинаковыми, сами операции могут отличаться **для каждого типа данных**. Также существуют общие математические принципы, которые применимы к операциям разных типов.
