#  Тип String

Тип String является моделью текстовой **строки**. У типа **нет ограничения на длину**. Тип String является сложным типом и реализован на языке Java, но полностью совместим с Scala.

Тип представляет собой **неизменяемый** перечень значений типа Char. Все методы типа String не изменяют строку, а создают **новую строку**.

## Сравнение строк

Строки можно сравнивать на равенство и больше или меньше. Строки сравниваются посимвольно, начиная с первого символа и до тех пор, пока не будет найдено отличие или одна из строк не закончится. Если есть разница в символах, то строка с символом с большим кодом будет большей. Если все символы совпадают, а одна строка длиннее другой, то более длинная строка считается большей. Пример сравнения строк:

val str1: String = "apple"
val str2: String = "banana"
val str3: String = "apple"
val str4: String = "apply"

str1 > str2 // false
str1 == str3 // true
str1 > str4 // false

##  Литералы типа String

### Простой строковый литерал

Литерал состоит из произвольного количества символов, заключенных в **двойные кавычки**. Пример: "abc"

Можно указывать все спец.символы, с помощью символа экранировки `\`. Пример: "\n"

Можно задать пустую строку: ""

### Литерал с интерполяцией s

Литерал позволяет встраивать значение произвольного типа в строку, без использования операции соединения строк.

val old = 25
val res = s"Have you turned $old yet?" // res будет иметь значение: Have you turned 25 yet?

Необходимо перед открывающейся кавычкой указать символ `s`, а месте, где в строке надо вставить значение, указать символ `$` и название значения.

Встраивать в строку также можно и выражения:

val old = 25
val add = 1
val res = s"Have you turned ${old + add} yet?" // res будет иметь значение: Have you turned 26 yet?

Выражение должно быть обернуто фигурными скобками.

### Литерал с интерполяцией f

Интерполятор строк f позволяет прикреплять к встроенным выражениям инструкции форматирования значения. Инструкции ставятся после выражения и начинаются со знака процента (%)

Инструкция использует синтаксис, определяемый классом java.util.Formatter. Данные инструкции рассмотрим позже.

Например, вот как можно отформатировать число π, ограничив количество дробных знаков только 5:

`val res = f"${math.Pi}%.5f" // res будет иметь значение 3,14159`

 

### Неформатированная строка

Неформатированная строка начинается и заканчивается тремя идущими подряд двойными кавычками `"""`. Внутри нее могут содержаться любые **специальные символы и двойные кавычки**, которые будут представлены в значении в том же виде, как и указаны.

Исключение составляет три двойные кавычки подряд `"""`.  Они обозначают конец строки.

Пример:

`val res = """Наберите "HELP" и '\n' для вызова помощи.""" // res будет иметь значение: Набирите "HELP" и '\n' для вызова помощи.`
