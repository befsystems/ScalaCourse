# Универсальная операция объединения

В типе String присутствует уникальная универсальная операция объединения (concatination) с именем **`+`** . Эта операция может **объединить** строку со значением **любого типа** данных. Результатом этой операции является значение с типом String.

Значение любого типа данных в этой операции **автоматически **преобразуется в строку и объединяется с заданной строкой.

Пример:

val res = "ab" + 1 // значение res будет иметь "ab1"
val res2 = 2L + "sd" // значение res2 будет иметь "2sd"
val res3 = 'a' + "sd" // значение res3 будет иметь "asd"
val res4 = "ab" + "sd" // значение res4 будет иметь "absd"

Из этого следует, что **любой **тип данных может быть преобразован в строку. Даже если это например тип видео-данных, то он неким образом должен преобразовываться в строку.

Приоритет операции - **высокий**, и она успешно **маскирует ошибки** математических операций.

Пример:

val one = "1"
val res = 1 + one // значение res будет иметь "11", а не 2 как скорее имелось ввиду

val one = "1".toInt
val res = 1 + one // значение res будет иметь 2

val one: Int = "1".toInt
val res: Int = 1 + one // не помешает явное указание типа
