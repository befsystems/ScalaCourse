# Выражение

**Выражение **(expression) - это последовательность операций, которая вычисляется в значение. Соответственно, **тип выражения** - это тип его вычисляемого значения.

В функциональной парадигме, выражения - это базовый строительный блок кода. Программа рассматривается как набор выражений, и итоговые результаты программы - это вычисление всех последовательностей выражений.

Пример простого выражения, состоящего из одной операции:

`1 + 2`

Значение выражения будет 3 типа Int.

Пример более сложного выражения:

`1 + 2 + (3 max 4) * 17`

Значение выражения будет 71 типа Int

Выражения можно соединять друг с другом с помощью любой доступной операции. Выражения, при этом, являются операндами соединяющей операции:

1 + 2 // выражение1
5 - 6 // выражение2
(1 + 2) * (5 - 6) // выражения соединяются вместе с помощью операции *

Значение выражения будет -3 типа Int

Теоретически, весь код программы функционального языка можно написать как одно большое выражение. Но это крайне непрактично, так как усложняет понимание и сопровождение программного кода. Необходимо делить выражения на логические части.

Разбиваем сложное выражение на части и вычисляем значение этих частей, и уже из этих значений создаем другое выражение.

val expression1: Int = 1 + 2 + (3 max 4) * 17
val expression2: Int = 5 + 6
val result: Int = expression1 * expression2

В итоге мы получили значение result равный 781 типа Int
