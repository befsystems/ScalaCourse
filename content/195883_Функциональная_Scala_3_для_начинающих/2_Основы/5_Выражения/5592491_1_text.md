# **Комментарий в коде программы**

**Комментарий** -- это текстовая часть исходного кода, которая не компилируется и не выполняется во время выполнения программы. Комментарии добавляются в код для пояснения, документации или временного исключения части кода из выполнения.

Примеры:

// Это однострочный комментарий
val x = 42 // Этот комментарий может идти и в конце строки кода

/*
Это многострочный комментарий.
Он может охватывать несколько строк.
*/
val y = 123
