# Логическая конструкция **if**

Конструкция **if** позволяет получить значение в зависимости от логического выражения.

if логическое_выражение then
// выражение, если логическое выражение истинно
else
// выражение, если логическое выражение ложно

Конструкция **if** является выражением.

Пример:

val a = 5
val b = 9
val res = if a > b then
a - b
else
b - a // вычисляется это выражение

// res имеет значение 4

Конструкцию можно написать в одну строчку:

`val res = if a > b then a - b else b - а`

Это конструкция может иерархически повторяться и иметь любую вложенность.

Пример вложенной конструкции:

val a = 5
val b = 10
val res = if a > b then
if (a - b) > 0 then
a * b
else
a + b
else
b - a // вычисляется это выражение

// res имеет значение 5

Все логические операции можно построить на конструкции `**if**`.

Пример реализации операции **Логическое И** на конструкции `if`:

val a = true // например
val b = false // например

// операция Логическое И
val res = if a then
b
else
false

// res = false

**Внимание**! Существует императивный вариант конструкции `**if**`, который может не использовать оператор `**else.**` Если вы забудете указать **`else`**в своей конструкции, то компилятор может создать императивный вариант конструкции, а он делает совсем не то, что задумывалось.
