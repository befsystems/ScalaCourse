# Операции типов Double и Float

Типы `Double` и `Float` имеют одинаковые операции.

В типах нет недопустимых комбинаций операндов и операций, вызывающих ошибки вычисления.

Типы имеют иерархию `Byte -> Short -> Int -> Long -> Float -> Double`. Это означает, что если в операции участвуют значения разных типов, то тип результата будет как у самого правого в цепочке.



## Арифметические операции

### Сложение

- **Обозначение операции:** `+`

- **Описание:** Выполняет сложение двух чисел

- **Тип:** Инфиксная операция (два операнда)

### Вычитание

- **Обозначение операции:** `-`

- **Описание:** Выполняет вычитание одного числа из другого

- **Тип:** Инфиксная операция (два операнда)

### Умножение

- **Обозначение операции:** `*`

- **Описание:** Выполняет умножение двух чисел

- **Тип:** Инфиксная операция (два операнда)

### Деление

- **Обозначение операции:** `/`** **

- **Описание:** Выполняет деление одного числа на другое

- **Тип:** Инфиксная операция (два операнда)

### Остаток от деления

- **Обозначение операции:** `%`

- **Описание:** Возвращает остаток от деления одного числа на другое. Операция для чисел с плавающей точкой работает по принципу взятия остатка от деления, как для целых чисел. Пример: 10.0 % 3.0 = 1.0

- **Тип:** Инфиксная операция (два операнда)

### Отрицание

- **Обозначение операции:** `-` (указывать перед значением)

- **Описание:** Меняет знак числа. Если исходное значение положительное, то после операции будет получено отрицательное значение, и наоборот.

## Операции отношения

### Равенство

- **Обозначение операции:** `==`

- **Описание:** Проверяет, равны ли два значения. Возвращает при успехе `true`, и в противном случае `false`

- **Тип:** Инфиксная операция (два операнда)

- Внимание! Не рекомендуется использовать операцию, из за возможных проблем с точностью представления значений.

### Неравенство

- **Обозначение операции:** `!=`

- **Описание:** Проверяет, не равны ли два значения. Возвращает при успехе `true`, и в противном случае `false`

- **Тип:** Инфиксная операция (два операнда)

- **Внимание! **Не рекомендуется использовать операцию, из за возможных проблем с точностью представления значений.

### Отношение больше

- **Обозначение операции:** `>`

- **Описание:** Проверяет, больше ли первое значение чем второе. Возвращает при успехе `true`, и в противном случае `false`

- **Тип:** Инфиксная операция (два операнда)

### Отношение меньше

- **Обозначение операции:** `<`

- **Описание:** Проверяет, меньше ли первое значение чем второе. Возвращает при успехе `true`, и в противном случае `false`

- **Тип:** Инфиксная операция (два операнда)

### Отношение больше или равно

- **Обозначение операции:** `>= `

- **Описание:** Проверяет, больше ли первое значение чем второе или равно. Возвращает при успехе `true`, и в противном случае `false`

- **Тип:** Инфиксная операция (два операнда)

- **Внимание!** Не рекомендуется использовать операцию, из за возможных проблем с точностью представления значений.

### Отношение меньше или равно

- **Обозначение операции:** `<= `

- **Описание:** Проверяет, меньше ли первое значение чем второе или равно. Возвращает при успехе `true`, и в противном случае `false`

- **Тип:** Инфиксная операция (два операнда)

## Операции преобразования

- `toByte` — преобразует число в тип `Byte`.

- `toShort` — преобразует число в тип `Short`.

- `toLong` — преобразует число в тип `Long`.

- `toFloat` — преобразует число в тип `Float`.

- `toDouble` — преобразует число в тип `Double`.

- `toChar` — преобразует число в символ `Char` по Unicode-коду.

- `toString` — преобразует значение в строку.

- `toHexString` — возвращает строковое представление числа в шестнадцатеричной системе.

- `toOctalString` — возвращает строковое представление числа в восьмеричной системе.

- `toBinaryString` — возвращает строковое представление числа в двоичной системе.

- `toDegrees` — преобразует угол из радианов в градусы.

- `toRadians` — преобразует угол из градусов в радианы.

- `round` — возвращает значение с округлением до ближайшего целого типа `Long`

- `abs`— возвращает положительное значение числа

Это постфиксные операции.

## Операции принадлежности значения

- `isNaN` - значение не число

- `isInfinity` - значение бесконечность

- `isFinite` - значение конечное число

- `isPosInfinity` - значение положительная бесконечность

- `isNegInfinity` - значение отрицательная бесконечность

Это постфиксные операции.

- **Тип:** Инфиксная операция (два операнда)

### Максимальное значение

- **Обозначение операции:** `max`

- **Описание:** Возвращает из двух значений максимальное. Только одинаковые типы

- **Тип:** Инфиксная операция (два операнда)
