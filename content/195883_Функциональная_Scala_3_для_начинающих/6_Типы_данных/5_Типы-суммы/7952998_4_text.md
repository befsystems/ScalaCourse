## Literal Types — точные значения как типы

**Literal Types**  — это типы, которые представляют конкретные значения, а не множества значений.

// Обычные типы — множества значений
val name: String = "Alice" // может быть любая строка
val age: Int = 30 // может быть любое число

// Literal types — точные значения
val method: "GET" = "GET" // может быть только "GET"
val port: 8080 = 8080 // может быть только 8080
val flag: true = true // может быть только true
val grade: 'A' = 'A' // может быть только 'A'

Literal types особенно мощны в комбинации с union types:

// ✅ Работает
type SimpleNumber = 42 | 84
type SimpleString = "hello" | "world"
type SimpleBool = true | false

// ❌ Не работает
// type ComplexLiteral = List(1,2,3) | Map("key" -> "value")

val specific: "GET" = "GET"
val general: String = specific // ✅ Widening работает
