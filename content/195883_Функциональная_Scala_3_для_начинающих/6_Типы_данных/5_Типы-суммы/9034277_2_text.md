# Алгебра типа-суммы

## Кардинальность типа-суммы

Кардинальность Sum type вычисляется как сумма кардинальностей его альтернатив:

**|T₁ + T₂ + ... + Tₙ| = |T₁| + |T₂| + ... + |Tₙ|**

**Примеры:**

// |"red" + "green" + "blue"| = 1 + 1 + 1 = 3
type Color = "red" | "green" | "blue"

// |true + false| = 1 + 1 = 2
type Bool = true | false

// |1 + 2 + 3 + 4 + 5| = 1 + 1 + 1 + 1 + 1 = 5
type Digit = 1 | 2 | 3 | 4 | 5

// |"small" + "medium" + "large"| = 1 + 1 + 1 = 3
type Size = "small" | "medium" | "large"

// |Size + Shape| = 3 + 3 = 6
type SizeOrShape = Size | Color
// Возможные значения: "small", "medium", "large", "red", "green", "blue"

## Математические свойства типа-суммы

Sum type обладают алгебраическими свойствами:

- **Коммутативность:** A + B ≅ B + A (изоморфны)

- **Ассоциативность:** (A + B) + C ≅ A + (B + C)

- **Нейтральный элемент:** A + Nothing ≅ A

где символ ≅ означает изоморфизм — типы можно однозначно преобразовать друг в друга.

Примеры преобразований:

/ A + B ≅ B + A
type AB = "a" | "b"
type BA = "b" | "a"

val ab: AB = "a"
val ba: BA = ab // Автоматическое преобразование

// (A + B) + C ≅ A + (B + C)
type ABC1 = ("a" | "b") | "c"
type ABC2 = "a" | ("b" | "c")

val abc1: ABC1 = "a"
val abc2: ABC2 = abc1

// A + ∅ ≅ A
type AWithEmpty = "a" | Nothing
type JustA = "a"

val withEmpty: AWithEmpty = "a"
val justA: JustA = withEmpty

### **Тип Nothing как нетральный элемент в алгебре** Sum type**:**

- `Nothing` соответствует нулю в алгебре типа (как 0 в сложении: 0 + x = x)

- В теории типов `Nothing` соответствует **пустому типу** (⊥), также называется **bottom type**

- `Nothing` — это тип **без значений** (uninhabited type), поэтому **значение** типа `A | Nothing` может быть только значением типа `A`
