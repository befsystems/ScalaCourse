# Операции кортежа

| Операция | Описание |
| --- | --- |
| (a1, ..., an)**  ==  **(b1, ..., bn) | Сравнение кортежей только **одинакового **размера. Если **тип или значение** хотя бы одного элемента не совпадает, то выдает значение **false**. В противном случае **true**. |
| (a1, ..., an)  **:***  b | Возвращает копию кортежа, с добавленным элементом в конце кортежа (a1, ..., an,  b) |
|  a  ***:**  (b1, ..., bn) | Возвращает копию кортежа, добавив элемент в начало кортежа  (a,** **b1, ..., bn). Операция занимает время - O(size) |
| (a1, ..., an) **++** (b1, ..., bm) | Возвращает новый кортеж, объединив два кортежа (a1, ..., an, b1, ..., bm) Операция занимает время O(a.size + b.size) |
| (a1, ..., an)**  size ** | Возвращает размер (или арность) кортежа |
| (a1, ..., an)**  head** | Возвращает голову кортежа a1 |
| (a1, ..., an)**  init** | Возвращает начальную часть кортежа (a1, ..., an-1) |
| (a1, ..., an)**  last** | Возвращает последний элемент кортежа an |
| (a1, ..., an)**  tail** | Возвращает хвост  кортежа (a2, ..., an) |
| (a1, ..., an)**  map** | Преобразовывает все элементы кортежа, используя полиморфную лямда-функцию |
| (a1, ..., an)** ** **zip  **(b1, ..., bm) | При двух кортежах, (a1, ..., an)** ** и (b1, ..., bm), возвращает кортеж ((a1, b1), ..., (an, bn)). Если два кортежа имеют разные размеры, дополнительные элементы большего кортежа будут проигнорированы. |
| (a1, ..., an) **take** m | Возвращает новый кортеж, с первыми m элементами (a1, ..., am) |
|  (a1, ..., an) **drop** m | Возвращает новый кортеж, с элементами кроме первых m  (am, ..., an) |
| (a1, ..., an) **splitAt **m | Возвращает два новых кортежа, используя элементы из старого (a1, ..., am) и (am+1, ..., an) |
| (a1, ..., an) **reverse** | Возвращает обратный кортеж (an, ..., a1) |
| (a1, ..., an) **toString** | Отображает кортеж в строку |
| (a1, ..., an) **_m** | Возвращает m-ый элемент кортежа am |
| (a0, ..., an-1)**(i)** | Возвращает элемент кортежа по индексу i,  ai     Индекс первого элемента - это 0. |
| val (x, ..., z) **=** (a1, ..., an) | Декомпозиция кортежа на отдельные элементы x, ..., z |
| a** -> **b | Создание кортежа двух элементов (a, b) |
