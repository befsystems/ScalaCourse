# Функциональные значения

**Функциональные значения** - это **значения функционального типа** (`A => B`, `(A, B) => C`, etc.), которые представляют собой **вычислительное поведение**, и в то же время могут использоваться как **значения**.

**Важно понимать,** что функциональное значение - это значение, которое обозначает не данные, а **вычисления**.

На самом деле, анонимная функция лишь синтаксическая конструкция, которая в системе типов значиться как еще одно функциональное значение.

В функциональное значение всегда преобразуются:

- Анонимные функции

- Частично примененные функции

- Каррированные функции

- Именованные функции - после специального преобразования

Именно функциональным значением оперирует программа, когда в коде программы ей встречается некая функциональная конструкция. Это обеспечивает единую математику в системе типов для всех конструкций.
