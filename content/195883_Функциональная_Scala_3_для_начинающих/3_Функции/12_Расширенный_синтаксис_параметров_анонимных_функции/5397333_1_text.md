# Не указывать типы параметров

В анонимной функции можно не указывать тип параметров при её определении, если **известен** **тип функционального значения**.

Пример:

`val fun: Char => Unit = param => println(param) // тип параметра param можно не указывать`

Тип функционального значения `Char => Unit`. Из этого типа можно вывести тип параметра - он будет `Char`. Можно не указывать тип параметра, он будет выведен автоматически.

Круглые скобки можно не указывать вокруг одиночного параметра без типа.

Но если функциональный тип не задан, то нужно обязательно указать тип параметра.

`val fun = (param: Char) => println(param) // параметр param имеет тип Char`

Зная тип параметра и тип выходного значения легко вычисляется тип функционального значения `Char => Unit`
