# Без определения параметров

В анонимных функциях можно вообще не определять параметры, а сделать это по мере их использования в теле функции. Но использование этих параметров должно отвечать следующим требованием:

- Все параметры функции должны быть использованы.

- Все параметры функции должны быть использованы только один раз.

- Использование параметров функции должно быть выполнено в порядке их следования в функциональном типе.

В месте использования параметра нужно указать знак подстановки _.

Пример:

`val fun: (Char, Int, String) => String = _ + _ + _ `

В данном примере под первым знаком подстановки будет использован первый параметр типа Char, под вторым знаком подстановки будет использован второй параметр типа Int, и под третьим знаком подстановки будет использован третий параметр типа String.

Если тип функционального значения не задан, то тип параметра необходимо указать явно при использовании.

`val fun = (_: Char) + (_: Int) + (_: String) `

Зная тип параметров и тип выходного значения легко вычисляется тип функционального значения (Char, Int, String) => String
