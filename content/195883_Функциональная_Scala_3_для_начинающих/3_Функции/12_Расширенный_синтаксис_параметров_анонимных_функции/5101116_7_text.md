# Скрытый параметр

**Одиночный** параметр функции можно совсем нигде не указывать, если он применяется как **аргумент** другой функции с **одним параметром** и только **один раз**. Компилятор сам подставит параметр в нужное место.

val fun: Char => Unit = char => println(char) // явное указание параметра
val fun: Char => Unit = println // функция println имеет один параметр, но он не указывается.

Данный прием используется для написания специальных лексических строковых конструкций. При обычном кодировании лучше его не использовать.
