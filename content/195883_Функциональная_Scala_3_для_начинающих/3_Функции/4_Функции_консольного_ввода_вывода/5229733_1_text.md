# Термины и определения

**Консоль** или командная строка, это текстовый командный интерфейс для взаимодействия с операционной системой и программами. Она предоставляет пользователю возможность вводить **команды** и получать **результаты** их выполнения в текстовом виде.

В **задачах данного курса**, консоль **имитируется** входным и выходным текстовыми полями, в которые можно вводить текст перед запуском задачи, и наблюдать выходной текст задачи.

# Функции консольного ввода/вывода

| Функция | Описание |
| --- | --- |
| def print(x: Any): Unit | Функция `print` выводит значения `x` на стандартный вывод. Функция выводит значение без добавления символа новой строки в конце вывода. Внутри функции `x` будет автоматически преобразован в строку для вывода. Тип параметра Any - означает что можно использовать любой тип. |
| def println(): Unit | Функция `println()` используется для вывода пустой строки на стандартный вывод. Она не принимает аргументов и просто выводит символ новой строки. |
| def println(x: Any): Unit | Функция `println(x: Any)` используется для вывода значения `x` на стандартный вывод, за которым следует символ новой строки. Внутри функции `x` будет автоматически преобразован в строку для вывода. Тип параметра Any - означает что можно использовать любой тип. |
| def readLine(): String | Функция `readLine()` используется для считывания строки из стандартного ввода (обычно это консоль). Она ожидает ввода пользователя и возвращает введенную строку в качестве результата. |
| def readBoolean(): Boolean | Функция `readBoolean()` используется для чтения значения типа `Boolean` из стандартного ввода. Она ожидает ввода пользователя в виде строки, которая будет интерпретирована как логическое значение `true` или `false`. Если введенная строка соответствует значению `true` (без учета регистра), то функция возвращает `true`, в противном случае возвращается `false`. |
| def readChar(): Char | Функция `readChar()` используется для чтения символа из стандартного ввода. Она ожидает ввода пользователя в виде строки, и возвращает первый символ этой строки. Если строка пуста, функция выдаст ошибку. |
| def readInt(): Int | Функция `readInt()` используется для чтения целого числа из стандартного ввода. Она ожидает ввода пользователя в виде строки, пытается преобразовать эту строку в целое число и возвращает результат. Если ввод пользователя не может быть преобразован в целое число, функция выдаст ошибку. |
| def readLong(): Long | Функция `readLong()` используется для чтения длинного целого числа (тип `Long`) из стандартного ввода. Она ожидает ввода пользователя в виде строки и пытается преобразовать эту строку в длинное целое число. Если ввод пользователя не может быть преобразован в `Long`, функция выдаст ошибку. |
| def readDouble(): Double | Функция `readDouble()` используется для чтения числа с плавающей точкой (тип `Double`) из стандартного ввода. Она ожидает ввода пользователя в виде строки и пытается преобразовать эту строку в число с плавающей точкой. Если ввод пользователя не может быть преобразован в `Double`, функция выдаст ошибку. |
