# Многострочное тело функции

Если тело функции состоит из **нескольких** выражений, то образуется блок выражений. Блок выражений обозначается фигурными скобками.

Пример:

def add(param1: Int, param2: Int): Int = {
val temp1 = param1 + 1
val temp2 = param2 - 1
temp1 + temp2
}

Можно **не использовать** фигурные скобки вообще, если отступы каждого выражения блока от левого края **одинаковые**.

Пример:

def add(param1: Int, param2: Int): Int =
val temp1 = param1 + 1
val temp2 = param2 - 1
temp1 + temp2
