# Именованные функции с одинаковыми именами

Именованные функции могут быть с **одинаковыми именами**. В этом случае у них должны **отличаться списки параметров**. Название параметров не имеет значения, но их типы должны представлять уникальную комбинацию в порядке определения.

Пример определения функций с одинаковыми именами:

def add(param1: Int, param2: Int): Int = ???
def add(param1: Double, param2: Int): Int = ???
def add(param1: Int, param2: Double): Int = ???

Это совершенно разные функции, хотя имя у них одинаковое.

Тип **выходного** значения никак **не влияет **на возможность определения таких функций.

Имя функции и список параметров образуют **сигнатуру** функции, которая должна быть уникальная.
