# Функции высшего порядка

**Фу́нкция вы́сшего поря́дка** (Higher-order functions) — функция, принимающая в качестве аргументов другие функции, или возвращающая другую функцию в качестве результата.

Пример функции высшего порядка и её использование:

def func(str: String, int: Int, func2: Int => String) = str + func2(int) // определение функции высшего порядка, где у третьего параметра функциональный тип.

func("1", 2, s => s.toString) // вызов функции высшего порядка, где третьим аргументом является анонимная функция

Использование функции, переданной через функциональный параметр, ничем не отличается от использования функции взятой не через параметр. Однако, функция полученная через параметр заранее неизвестна, она может быть любой подходящей по типу, а функция указанная в теле родительской функции непосредственно, уже не может быть изменена при выполнении.

Назначение функций высокого порядка - это сокращение объема кода и увеличение универсальности функции за счет разделения функции на фиксированную часть и изменяемую часть. Фиксированная часть помещается в самой функции, а изменяемая часть передается функции как аргумент.

Функции высокого порядка позволяют работать с абстракциями данных, такими как списки, множества и деревья, без необходимости знать их внутреннее представление.

Функциями высокого порядка могут быть как именованные функции, так и анонимные функции.

Количество функциональных параметров у функции может быть любым. Однако большое их количество сильно усложняет читаемость и понимание функций.

## Функция - как создатель функций

Функции высокого порядка позволяют генерировать другие функции в процессе выполнения программы, в зависимости от получаемых данных. Это позволяет создавать универсальные и очень гибкие алгоритмы.

Пример функции высшего порядка с функциональным выходным значением:

// Определение функции - генератора функций
def algorithmGen(isFirstAlgorithm: Boolean): Int => String =
if (isFirstAlgorithm)
(int2: Int) => (int2 * 2).toHexString
else
(int2: Int) => (int2 * 3).toHexString

// Используем функцию algorithmGen для создания функций firstAlgorithm и secondAlgorithm
val firstAlgorithm: Int => String = algorithmGen(true)
val secondAlgorithm: Int => String = algorithmGen(false)

firstAlgorithm(2) // Результат "4"
firstAlgorithm(3) // Результат "6"

secondAlgorithm(2) // Результат "6"
secondAlgorithm(3) // Результат "9"
