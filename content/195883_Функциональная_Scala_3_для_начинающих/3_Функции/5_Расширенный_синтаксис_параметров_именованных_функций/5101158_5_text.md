# Список аргументов переменной длины

При вызове функции можно указывать список аргументов **переменной** длины, если **последним** параметром в списке указать **повторяющийся** параметр.

Пример:

def add(param1: Int, param2: Char*): String = param1.toString + param2.mkString
val res = add(1, 'a', 'b') // Результат строка "1ab"
val res2 = add(1, 'a', 'b', 'c') // // Результат строка "1abc"
val res3 = add(1) // // Результат строка "1"

где **Char*** - повторяющийся параметр,

К типу Char добавлена звездочка *****, которая указывает, что параметр может принимать список аргументов типа Char.

Операция **mkString** создает строку из списка символов. В теле функции с повторяющимся параметром нужно работать как со списком значений. Работа со списками будет освещена в будущих занятиях.
