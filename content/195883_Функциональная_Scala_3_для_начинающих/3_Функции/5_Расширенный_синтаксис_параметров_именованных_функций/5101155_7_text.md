# Аргумент функции - блок выражений

Если у функции имеется только **один параметр**, то при вызове такой функции, аргументом можно указать сразу **блок выражений**.

Пример:

def inc(param: Int): Int = param + 1 // функция inc с одним параметром

// Вариант с аргументом значением
val a = 1 + 1
val b = a + 3
val res1 = inc(b) // вызов функции inc с аргументом - значением. Результат 6

// Вариант с аргументом - блоком выражений
val res2 = inc { // начало блока выражений
val a = 1 + 1
a + 3
} // конец блока выражений
// Результат 6

Блок выражений всегда имеет одно значение, которое и будет передано в тело функции. Тип блока выражений должен совпадать с типом параметра функции.

Такой синтаксис позволяет создавать более выразительные вложенные композиции, с минимальным количеством скобок и промежуточных значений.
