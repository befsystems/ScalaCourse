# Главная функция программы

**Главной функцией** называется функция, с вызова которой начинается выполнение программы. Её также называют **точкой входа** в программу.

В Scala 3 роль главной функции выполняет любая функция, помеченная аннотацией `@main`.

#### Основные требования к главной функции:




Должна быть определена с помощью `def`;




Должна возвращать `Unit`;




Должна быть помечена аннотацией `@main`;




Может иметь любое имя;




Может принимать любое количество параметров, включая переменное количество (`String*`);




Может располагаться в любом месте с **глобальным (статическим) доступом**;




В одной программе может быть **несколько** главных функций.


Пример:

@main def run(): Unit =
println("Program running!")

@main def run2(params: String*): Unit =
println(s"Program running! Arguments: ${params.mkString(", ")}")

@main def run3(param1: Int, param2: String): Unit =
println(s"Program running! Arguments: $param1, $param2")



**При запуске программы** имя нужной функции указывается в командной строке. Файл с функциями в примере называется `Main.scala`

Пример запуска программы:

`scala-cli Main.scala -- run2 Hello Scala `

Вывод:

`Program running! Arguments: Hello, Scala`
