# Вызов именованной функции

Использование или вызов именованной функции осуществляется с использованием её имени и с указанием аргументов для её параметров.

Пример:

def add(param1: Int, param2: Double): Double = ??? // определение функции add
val res: Double = add(1, 2.1) // вызов (использование) функции add

Результат функции будет содержать значение res.

Аргументы для параметров должны быть указаны через запятую, в том порядке, как были определены параметры.

Тип подставляемых аргументов должен соответствовать типу параметров.



Аргументами могут выступать **все виды** выражений и значений соответствующего типа.

Это могут быть литералы:

val res: Double = add(1, 2.1)

Это могут быть значения:

val param = 1
val res: Double = add(param, 2.1)

Это могут быть сложные выражения:

val res: Double = add(1 + 1, 2.0 + 0.1)

Это могут быть вызов функции:

val res: Double = add(1, add(1, 1.0), 2.0)
