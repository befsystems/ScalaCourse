# Обратная функция

**Обратная функция** — это такая функция, которая "отменяет" действие другой функции. Если у нас есть функция *`f`*, которая преобразует элемент *`x`* в элемент *`y`* (то есть \(f(x)=y\)), то **обратная функция** \(f^{-1}\) преобразует *`y`* обратно в *`x`*, то есть \(f^{-1}(y) = x\).

Говоря более формально, функция *`f`* имеет обратную функцию \(f^{-1}\), если для любого `*x*` из области определения *`f`*:

\(f(f^ {−1} (y))=y\)   и   \(f ^{−1} (f(x))=x\)

Это означает, что если мы сначала применим функцию `*f*`, а затем её обратную функцию\(f^{-1}\), мы вернёмся к исходному значению. И наоборот: если сначала применить\(f^{-1}\), а затем `*f*`, результатом будет исходное значение.

### Пример обратной функции

Рассмотрим функцию \(f(x) = x + 2\). Чтобы найти обратную функцию \(f^{-1}\), нам нужно решить уравнение для `*x*`:

\(f(x) = y \quad \Rightarrow \quad x + 2 = y \quad \Rightarrow \quad x = y - 2\)

Таким образом, обратная функция \(f^{-1}(y) = y - 2\)

### Свойства




Функция может не иметь обратной функции.




Функция и обратная функция могут быть взаимообратными.
