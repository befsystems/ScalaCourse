# Функции в Scala

Функциями в Scala можно назвать конструкции выполняющие **преобразование **(отображение) значений одного типа данных в единственное значение того же или другого типа данных.

Функция — это особая конструкция **выражения**, которая использует **параметры**.

Функции в Scala представлены двумя видами, это **именованная функция** и **анонимная функция** (лямбда-функция). Причина наличия в языке двух видов функций, связана с реализацией полноценной поддержки одновременно двух концепций программирования - объектно-ориентированой и функциональной.

Функции могут быть **высшего порядка** (Higher-Order Functions). Это позволяет передавать одну функцию для использования другой функции, или возвращать одну функцию как результат другой функции. Функция может создать другую функцию.

Функции могут быть **частично определенными** (Partial Functions). Частично определённая функция - это функция, которая определена не для всех возможных значений из её области определения. Она может вернуть результат только для некоторых значений входного аргумента.
