# Функция

**Фу́нкцией** в математике называется **соответствие между двумя множествами**, при котором каждому элементу одного множества (область определения) ставится в соответствие **единственный** элемент другого множества (множество значений).

Также, функцией называют **отображение** одного множества в другое.

Пример функции в математике:

y = sin(x)

![Изображение](https://ucarecdn.com/9ff82510-f3d2-4be8-a4fb-c9ec0f8f599b/) Пример функции, сопоставляющей каждой из четырёх фигур её цвет. Можно записать как X -> Y.

У функции есть **входные** значения, **одно выходное** значение и **алгоритм ** отображения.

В программировании, функция представляет собой **блок кода**, который выполняет вычисление или некие действия. Функции принимают входные данные, выполняют определенные операции и возвращают результат. Функции являются важным строительным блоком в программировании и обеспечивают модульность, повторное использование кода и структурирование программы.

В функциональном стиле, функция – это **ведущая** конструкция программы. Все остальные конструкции и элементы обеспечивают комфортность работы с функциями.

Популярность функции из всех других математических конструкций связана с её удобными свойствами использования в вычислениях. Позднее мы разберем полезные свойства функций.

В функциональном стиле программирования ценится принцип декомпозиции программы на множество небольших функций, каждая из которых имеет четко определенную задачу. Эти функции обычно **невелики** по размеру, что обеспечивает **простоту **их понимания. Главное преимущество такого подхода заключается в том, что программисту предоставляется обширный набор строительных блоков, которые могут быть гибко **комбинированы** для решения более сложных задач. Эти блоки созданы таким образом, чтобы каждый из них был легко понятен в отдельности, что облегчает сопровождение и модификацию кода.

## Параметры функции

**Параметры функции** - это заранее **неизвестные** данные, описываемые при определении функции как входные значения функции. Параметры функции используются в теле функции в выражениях. Их количество может быть не ограничено.

## Аргументы функции

**Аргументы функции** - это **известные** значения, подставляемые в параметры функции при её вызове (вычислении).
