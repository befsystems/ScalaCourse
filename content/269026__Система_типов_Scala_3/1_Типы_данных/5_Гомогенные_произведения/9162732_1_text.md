# Гомогенные произведения

Гомогенные произведения (Homogeneous Products) — это типы данных, которые содержат **множество значений одного и того же типа**. В отличие от гетерогенных произведений (например, case class с полями разных типов), здесь все элементы имеют идентичный тип.

Обозначение: **T^n**, где T — базовый тип, n — количество элементов.

**Характерная особенность** представителей этих типов - фиксированное количество элементов на протяжении всего времени существования значения данного типа. 

## Кардинальность гомогенных произведений

Кардинальность показывает, сколько различных значений может принимать тип:

**`|T^n| = |T| × |T| × ... × |T| = |T|^n`**

Примеры вычисления кардинальности:

// Boolean^2 - пара булевых значений
type BoolPair = (Boolean, Boolean)
// |BoolPair| = |Boolean|^2 = 2^2 = 4
// Возможные значения: (true, true), (true, false), (false, true), (false, false)

// Boolean^3 - тройка булевых значений
type BoolTriple = (Boolean, Boolean, Boolean)
// |BoolTriple| = 2^3 = 8

// Byte^2 - пара байтов
type BytePair = (Byte, Byte)
// |BytePair| = 256^2 = 65,536

## Кортежи как гомогенные произведения

Кортежи могут быть гомогенными, когда все элементы одного типа:

// Гомогенный кортеж
val coordinates: (Int, Int, Int) = (10, 20, 30)
// |coordinates| = |Int|^3

## Основные представители

### Тип Array - Массив 

// Array[Boolean] длины 3
val flags: Array[Boolean] = Array(true, false, true)
// |flags| = 2^3 = 8 возможных комбинаций

### Тип String - Строка

// String длины 3
val code: String = "ABC"
// |code| = |Char|^3 = 65,536^3 = 281,474,976,710,656

### Тип Range - Диапазон чисел 

// Ленивая коллекция, с вычисляемой длиной
val r1 = Range(1, 5) // 12345

###  

## Ключевые алгебраические различия

*(справочно)*

| Свойство | Гетерогенное произведение | Гомогенное произведение |
| --- | --- | --- |
| **Коммутативность** | A × B ≠ B × A | T × T ≅ T × T (изоморфны) |
| **Показательная запись** | Нет | T^n |
| **Законы экспонент** | Не применимы | T^n × T^m = T^(n+m) (T^n)^m = T^(n×m) |
| **Нейтральный элемент** | Unit (1) | T^0 = 1 |
| **Map/Fold/Reduce** | Нет (но появляются) | Есть |
| **Перестановки** | Бессмысленны | Группа S_n |
| **Конкатенация** | Нет | Моноид |
| **Изоморфизмы** | Мало | Много (Array ≅ List ≅ Vector ≅ функции) |
| **Индексация** | По имени поля | По номеру (Int) |

**Главное отличие**: гомогенные произведения обладают **дополнительной симметрией** и **регулярной структурой**, что открывает богатую алгебру операций, недоступную для гетерогенных произведений.

## **Использование**

Использование типа обусловлено необходимостью иметь однородный набор значений фиксированного, неизменяемого размера:

- векторы,

- матрицы,

- координаты,

- битовые маски,

- цветовые модели
