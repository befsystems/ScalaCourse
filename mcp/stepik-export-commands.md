# StepikMCPServer - Работа с курсом через Claude Code

## О MCP сервере

StepikMCPServer предоставляет доступ к курсам Stepik через Model Context Protocol (MCP).
Сервер автоматически подключается к Claude Code и предоставляет инструменты для работы с курсами.

## Доступные инструменты

### 1. stepik_list_courses
Получение списка доступных курсов с пагинацией.

### 2. stepik_get_course
Получение детальной информации о курсе по ID.

### 3. stepik_get_course_structure
Получение структуры курса с разделами и уроками.

### 4. stepik_export_course
Экспорт полного содержимого курса в файлы.

## Использование через Claude Code

Все операции выполняются через естественные запросы к Claude:

**Примеры команд:**

```
Экспортируй курс Scala (ID 195883) в папку course
Покажи информацию о курсе 195883
Получи структуру курса 195883
Список доступных курсов
```

Claude автоматически использует нужные MCP инструменты для выполнения запроса.

## Формат файлов

- `.step` - Оригинальный JSON из API Stepik
- `.md` - Конвертированный Markdown для удобного чтения

## Конфигурация

Настройки сервера встроены в `StepikMCPServer-assembly-1.0.12.jar`:
- API credentials (clientId, clientSecret)
- Базовая директория экспорта: `baseDir = "../../course"`
- Формат: markdown
- Синхронизация: bidirectional

## Технические детали

MCP сервер настроен в `.mcp.json` в корне проекта и автоматически запускается при работе с Claude Code.
Путь к jar-файлу указан относительно корня проекта для переносимости конфигурации.
