# Индексы курсов Scala 3

Навигационная система для курсов Scala 3, загруженных со Stepik.

## Структура индексов

Индексы организованы по курсам. Каждый курс имеет собственный подкаталог с полным набором индексных файлов.

```
indexes/
├── 195883_Функциональная_Scala_3_для_начинающих/
│   ├── module-hierarchy.md      - иерархическая навигация
│   ├── topics-keywords.md       - тематический указатель
│   ├── quick-reference.md       - табличная навигация
│   ├── by-content-type.md       - группировка по типам
│   └── README.md                - описание индексов курса
│
└── 269026_Система_типов_Scala_3/
    ├── module-hierarchy.md      - иерархическая навигация
    ├── topics-keywords.md       - тематический указатель
    ├── quick-reference.md       - табличная навигация
    ├── by-content-type.md       - группировка по типам
    └── README.md                - описание индексов курса
```

## Типы индексных файлов

Каждый курс имеет 4 типа индексов + описание:

### 1. module-hierarchy.md
Иерархическая навигация по структуре курса:
- Модули → Уроки → Шаги
- Детальная информация о каждом шаге
- Темы и ключевые слова
- Статистика по урокам

### 2. topics-keywords.md
Тематический алфавитный указатель:
- Поиск по ключевым словам
- Алфавитная организация
- Ссылки на все упоминания термина

### 3. quick-reference.md
Табличная навигация:
- Компактная таблица всех шагов
- Быстрый поиск по ID или позиции
- Общая статистика курса

### 4. by-content-type.md
Группировка по типам контента:
- Теория (text)
- Практика (code)
- Тесты (choice, matching)
- Статистика по типам

### 5. README.md
Описание индексной системы конкретного курса с примерами использования.

## Использование индексов

### Поиск по теме
1. Откройте нужный курс
2. Используйте `topics-keywords.md`
3. Найдите термин в алфавитном указателе

### Обзор структуры курса
1. Откройте нужный курс
2. Используйте `module-hierarchy.md`
3. Изучите иерархию модулей и уроков

### Быстрый поиск по ID
1. Откройте нужный курс
2. Используйте `quick-reference.md`
3. Найдите шаг по ID (Ctrl+F)

### Фильтрация по типу контента
1. Откройте нужный курс
2. Используйте `by-content-type.md`
3. Выберите нужный тип (text, code, choice, matching)

## Обслуживание индексов

### Когда обновлять индексы

Индексы нужно обновить при:
- Добавлении нового контента в курс
- Изменении структуры курса
- Загрузке нового курса со Stepik

### Как обновить индексы

Используйте команду индексации из модульной системы промптов:

```
@index-course [ID курса]
```

Система автоматически:
- Проанализирует структуру курса
- Извлечет метаданные
- Создаст/обновит все индексные файлы
- Проверит корректность путей

## Принципы организации

1. **Изоляция:** Каждый курс имеет собственный подкаталог индексов
2. **Консистентность:** Одинаковая структура индексов для всех курсов
3. **Компактность:** Индексы оптимизированы по размеру (цель: до 100 KB на файл)
4. **Навигация:** Множественные способы доступа к контенту
5. **Актуальность:** Автоматическая генерация с актуальными метаданными

## Связанные файлы

- **Конфигурация:** `stepik-config.json` - информация о всех загруженных курсах
- **Контент:** `content/` - исходные файлы курсов
- **Промпты:** `prompts/commands/index-course.md` - команда индексации
