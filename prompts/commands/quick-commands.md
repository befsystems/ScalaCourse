# Быстрые команды для Claude Code

## @prompt-architect

Активация роли архитектора промптов:

```
@prompt-architect: Используй роль из prompts/commands/prompt-architect-activation.md
```

## @prompt-create

Активация режима создания промптов:

```
@prompt-create: Используй роль из prompts/commands/prompt-creating.md
```

## Альтернативные короткие команды

**Для @prompt-architect:**

```
Переключись в режим архитектора промптов
```

```
Режим: архитектор промптов
```

**Для @prompt-create:**

```
Переключись в режим создания промптов
```

```
Режим: создание промптов
```

---

## Команды для работы с модулями промптов

```
@compose-prompt: [композиция промпта из модулей]
@create-module: [создание нового модуля]
```

## Команды для работы с планами

```
@create-plan: [создание плана работ]
@update-plan: [обновление плана]
@review-plan: [ревью плана]
@archive-plan: [архивация завершенного плана]
```

## Команды для работы с задачами

```
@create-task: [создание новой задачи]
@start-task: [начать работу над задачей]
@complete-task: [завершить задачу]
@block-task: [заблокировать задачу]
@update-task: [обновить статус задачи]
@show-kanban: [показать kanban-доску]
@archive-done-tasks: [архивировать выполненные]
```

## Команды для работы с индексацией курса

```
@index-course: [запуск агента индексации контента курса]
```

**Использование:**
- Команда автоматически запускает отдельного агента
- Агент обрабатывает весь курс поэтапно
- Результат: 4 индексных файла в директории indexes/
