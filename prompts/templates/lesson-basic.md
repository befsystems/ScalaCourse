# Шаблон: Базовый урок для начинающих

## Описание

Этот шаблон используется для создания базовых уроков курса Scala для начинающих программистов.

## Включаемые модули

### Обязательные модули
- `prompts/modules/common/context.md` - контекст проекта
- `prompts/modules/common/code-style.md` - стиль кода
- `prompts/modules/lesson/structure.md` - структура урока
- `prompts/modules/lesson/examples.md` - требования к примерам

### Рекомендуемые модули
- `prompts/modules/pedagogy/progression.md` - прогрессивное усложнение

## Инструкции по композиции

При использовании этого шаблона:

1. **Загрузить все обязательные модули** и применить их требования
2. **Определить специфику урока**:
   - Тема урока
   - Целевые концепции
   - Связь с предыдущими уроками
   - Предварительные требования
3. **Применить структуру** из `lesson/structure.md`
4. **Создать примеры** согласно `lesson/examples.md`
5. **Следовать прогрессии** из `pedagogy/progression.md`

## Специфические инструкции для начинающих

### Уровень детализации
- Объяснять даже базовые концепции
- Не предполагать знания других языков программирования
- Добавлять аналогии из повседневной жизни

### Язык изложения
- Простые предложения
- Избегать жаргона без объяснений
- Давать определения всем терминам

### Примеры кода
- Максимально простые (3-5 строк для базового примера)
- С детальными комментариями
- Показывать вывод для каждого примера

### Упражнения
- Четко сформулированные задачи
- Подсказки для решения
- Детальные тесты с понятными сообщениями об ошибках

## Формат вывода

```markdown
# [Название урока]

## Введение
[Мотивация, зачем это нужно, что узнаете]

## [Название концепции]
[Теоретическое объяснение]

### Синтаксис
[Базовый синтаксис с объяснением]

## Примеры

### Пример 1: [Базовое использование]
[Простейший пример]

### Пример 2: [Расширение]
[Чуть сложнее]

### Пример 3: [Практический пример]
[Реальный use case]

## Распространенные ошибки
[2-3 типичные ошибки с объяснением]

## Лучшие практики
[Рекомендации по использованию]

## Практическое задание
[Описание задачи]
[Входные данные]
[Ожидаемый результат]
[Подсказки]
[Тесты]

## Резюме
[Ключевые выводы]
[Ссылка на следующий урок]
```

## Пример использования

### Входные параметры
```
Тема: Option и работа с отсутствующими значениями
Аудитория: Начинающие
Предыдущий урок: Pattern matching
Следующий урок: Either и обработка ошибок
```

### Процесс композиции

Claude загрузит модули и создаст урок:
1. Применит контекст (Scala 3.3.1, Stepik, русский язык)
2. Использует стиль кода (идиоматичный Scala 3, читаемый код)
3. Следует структуре (введение → теория → примеры → ошибки → практика → резюме)
4. Создаст 3-4 примера (базовый → усложнение → практический)
5. Обеспечит прогрессию (каждый пример добавляет одну концепцию)

## Критерии качества

После создания урока проверить:
- [ ] Все секции из структуры присутствуют
- [ ] Примеры идут от простого к сложному
- [ ] Код компилируется и следует стилю
- [ ] Есть мотивация в начале
- [ ] Практическое задание с тестами
- [ ] Объем 30-45 минут (800-1500 слов)
- [ ] Язык доступен для начинающих
