# Промпты для разработки курса

Эта папка содержит промпты для Claude, используемые при создании материалов курса по Scala.

## Структура

### `lessons/`
Промпты для написания уроков. Здесь хранятся инструкции и шаблоны для создания структурированных уроков курса.

### `exercises/`
Промпты для создания упражнений. Содержит шаблоны для разработки практических заданий и задач для студентов.

### `theory/`
Промпты для теоретических материалов. Используются для написания объяснений концепций, документации и справочных материалов.

### `templates/`
Шаблоны промптов для повторного использования. Базовые промпты, которые можно адаптировать для различных целей.

### `agents/`
Промпты с правилами для суб-агентов. Содержит инструкции и настройки поведения для специализированных агентов, выполняющих различные задачи в процессе разработки курса.

## Использование

1. Сохраняйте промпты в формате `.md`
2. Используйте описательные имена файлов (например, `lesson-structure-prompt.md`)
3. Документируйте назначение каждого промпта в начале файла
4. Версионируйте промпты в Git для отслеживания изменений

## Рекомендации

- Пишите промпты на русском языке для удобства работы
- Включайте примеры ожидаемого результата
- Указывайте контекст и требования к выходному формату
- Обновляйте промпты на основе опыта использования
