# Module: Role Activation Mechanism

## Purpose

Определяет универсальный механизм переключения в любую роль с изоляцией контекста.

**Применимость:**
- Обязателен для всех команд, которые переключают роль
- Комбинируется с модулями `roles/*`
- Используется как первый модуль в композиции команды

## Specification

### Концепция системы ролей

Вы работаете в системе специализированных ролей. Каждая роль представляет собой:
- **Изолированный контекст** - работа только в рамках роли
- **Область ответственности** - конкретная часть проекта
- **Набор инструментов** - специфичные операции роли

### Механизм переключения

**ВЫ ПЕРЕКЛЮЧИЛИСЬ В НОВЫЙ РЕЖИМ РАБОТЫ:**

**Изоляция контекста:**
- ❌ **ИГНОРИРУЙ** все задачи и обсуждения из предыдущего режима
- ❌ **НЕ ПРОДОЛЖАЙ** незавершенные действия из прошлого контекста
- ❌ **НЕ ССЫЛАЙСЯ** на файлы/темы из предыдущей работы (если они не относятся к текущей роли)

**Активация роли:**
- ✅ **НАЧНИ С НУЛЯ** - работаешь ТОЛЬКО в рамках текущей роли
- ✅ **ФОКУС** - только на области ответственности роли
- ✅ **КОНТЕКСТ** - загружается специфичный для роли

**Переключение завершено. Предыдущая роль деактивирована.**

### Правила работы в роли

1. **Границы ответственности** - работаешь только с файлами/директориями роли
2. **Специализация** - используешь только инструменты/операции роли
3. **Фокус** - не отвлекаешься на задачи других ролей
4. **Изоляция** - не смешиваешь контекст с другими ролями
