# Модуль: Контекст проекта

## Назначение
Определяет базовый контекст проекта курса Scala для всех типов промптов.

## Применимость
- Тип промпта: все типы
- Обязательный: да
- Комбинируется с: любыми модулями

## Содержание

### Технический контекст
- **Язык**: Scala 3.3.1
- **Платформа**: Stepik
- **Build tool**: SBT 1.9.7
- **Формат материалов**: Markdown + JSON (Stepik format)

### Целевая аудитория
Курс предназначен для трех уровней:
- **Начинающие**: люди с базовыми знаниями программирования, начинают изучать Scala
- **Средний уровень**: программисты с опытом в других языках, переходят на Scala
- **Продвинутые**: опытные Scala разработчики, изучают продвинутые возможности

### Язык изложения
- Все материалы на **русском языке**
- Технические термины даются с английскими эквивалентами
- Стиль: практический, с акцентом на применение

### Формат курса
- Теоретическая часть (текст + примеры)
- Интерактивные примеры кода
- Практические упражнения с автотестами
- Проверка понимания через квизы

## Примеры использования

При создании урока:
```
Учитывай, что студент использует Scala 3.3.1 и формат Stepik.
Целевая аудитория: начинающие программисты.
Все материалы на русском языке.
```
