# CLAUDE.md

Этот файл предоставляет руководство для Claude Code (claude.ai/code) при работе с кодом в этом репозитории.

## Обзор проекта

Это проект разработки курса по Scala для платформы Stack, использующий Scala 3.3.1 и SBT 1.9.7. Проект служит комплексным учебным ресурсом и содержит практические примеры, упражнения и документацию для обучения программированию на Scala.

Проект следует стандартной структуре директорий SBT с исходным кодом в `src/main/scala/` и тестами в `src/test/scala/`.

### Цели курса

- Создание структурированных учебных материалов по программированию на Scala
- Разработка практических примеров кода, демонстрирующих концепции Scala
- Создание упражнений и тестов для практики студентов
- Документирование возможностей и лучших практик Scala
- Предоставление полной референсной реализации для курса на платформе Stack

## Основные команды

### Сборка и запуск
```bash
sbt compile    # Компиляция проекта
sbt run        # Запуск основного приложения
sbt clean      # Очистка артефактов сборки
```

### Тестирование
```bash
sbt test       # Запуск всех тестов
sbt testOnly TestClassName  # Запуск конкретного тестового класса
```

### Разработка
```bash
sbt ~compile   # Непрерывная компиляция при изменении файлов
sbt ~run       # Непрерывный запуск при изменении файлов
sbt ~test      # Непрерывное тестирование при изменении файлов
```

### REPL
```bash
sbt console    # Запуск Scala REPL с classpath проекта
```

## Структура проекта

- `build.sbt` - Главный файл конфигурации сборки
- `project/` - Файлы определения сборки SBT
- `src/main/scala/` - Основной исходный код Scala
- `src/test/scala/` - Тестовый исходный код
- `target/` - Директория вывода сборки (генерируется автоматически)
- `course/` - **Материалы курса для платформы Stack**
- `prompts/` - **Промпты для Claude при написании материалов курса**

### Структура курса

Курс представляет собой набор текстовых файлов, хранящихся в папке `course/` в корне проекта:

- **Файлы `.step`** - Оригинальные тексты в формате JSON, загруженные с платформы Stepik
- **Файлы `.md`** - Сконвертированный текст из одноименных файлов `.step` для удобства просмотра

Файлы курса распределены по каталогам уроков. Каждый каталог назван именем соответствующего урока и содержит все материалы (файлы `.step` и `.md`) для этого урока.

**Загрузка материалов курса:**
Файлы курса загружаются по команде разработчика непосредственно с платформы Stepik с помощью MCP сервера `StepikMCPServer`.

Эти файлы формируют структурированный курс обучения Scala на платформе Stepik.

### Структура промптов

Промпты для Claude при написании статей и материалов курса хранятся в папке `prompts/` в корне проекта:

- `prompts/lessons/` - Промпты для написания уроков
- `prompts/exercises/` - Промпты для создания упражнений
- `prompts/theory/` - Промпты для теоретических материалов
- `prompts/templates/` - Шаблоны промптов для повторного использования
- `prompts/agents/` - Промпты с правилами для суб-агентов

**Формат и организация:**
- Все промпты сохраняются в формате `.md` для удобства чтения и редактирования
- Файлы именуются описательно (например, `lesson-structure-prompt.md`, `exercise-creation-prompt.md`)
- Промпты версионируются в Git как важная часть процесса разработки курса
- Промпты документируют подход к созданию учебных материалов и обеспечивают консистентность

## Основное приложение

Точка входа находится в `src/main/scala/Main.scala` с аннотацией `@main`. Этот код демонстрирует различные возможности Scala 3 и служит примером материала для курса.
